# Blog Mapper Test

Proyecto Next.js con TypeScript y Tailwind CSS.

## Tecnologías

- **Next.js 16.1.6** - Framework React con App Router
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Framework CSS para estilos
- **ESLint** - Linter para código limpio

## Comenzar

Ejecuta el servidor de desarrollo:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

La página principal se encuentra en `src/app/page.tsx` y se actualiza automáticamente al editar.
